# Merchantise
## Aplicativo de mercado

### Funções:
* Tela de login para administrador e vendedor
* Cadastro de Vendedores
* Cadastro de Categoria de produto
* Cadastro de produtos
* Cadastro de Vendas
* Calculo do valor de vendas
* Impressão de PDF das vendas
